# LLM Classification Pipeline

## Overview
Built a two-stage LLM pipeline using GPT-4o and DeepSeek R1 to classify unstructured text.

## Features
- Structured data extraction
- Reasoning-based classification
- JSON intermediate outputs

## Tech Stack
Python, LLM APIs, prompt engineering

## Results
Evaluated using accuracy, AUC, and F1 score

## Notes
Designed to reduce domain bias and improve generalizability
